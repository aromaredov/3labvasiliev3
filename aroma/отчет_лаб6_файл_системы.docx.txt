4. Типы файлов в каталоге /dev
Команда ls -l /dev вывела список файлов. В каталоге /dev присутствуют следующие типы файлов:

Символ	Тип файла	Пример
d	Каталог	block, char, disk, pts
l	Символьная ссылка	core -> /proc/kcore, fd -> /proc/self/fd, stdin -> /proc/self/fd/0
c	Символьное устройство	null, zero, tty, console, random
b	Блочное устройство	sda, sdb, loop0, ram0
s	Сокет	log (может присутствовать в /dev)
p	Именованный канал	(в выводе не обнаружен, но может быть)

5. Изучение ключей утилиты ls (справка man ls)
С помощью команды man ls были изучены следующие ключи:

Ключ	Назначение
-R	Рекурсивный обход подкаталогов
-1	Вывод в одну колонку (единица)
-m	Вывод через запятую
--color	Цветная подсветка типов файлов

-S	Сортировка по размеру (большие сверху)
-X	Сортировка по расширению
-r	Обратный порядок сортировки

Ответы на контрольные вопросы
Типы файлов ОС Linux: обычные файлы (-), каталоги (d), символьные ссылки (l), блочные устройства (b), символьные устройства (c), сокеты (s), именованные каналы (p).

Назначение утилиты file: определяет тип файла по его содержимому (сигнатурам), не доверяя расширению имени.

Структура дерева каталогов ОС Linux: иерархическая, с единым корнем /, от которого отходят все подкаталоги.

Отличия файловых систем Windows и Linux: в Linux единый корень / (нет дисков C:), регистрозависимость имён, расширения не обязательны.

Отличие каталогов /var и /tmp: /var содержит постоянно изменяемые данные (логи, очереди), /tmp – временные файлы, очищается при загрузке.

Назначение утилиты pwd: выводит полный путь текущего каталога.

Назначение утилиты cat: выводит содержимое файла на экран (или объединяет несколько файлов).

Назначение утилиты ls: вывод списка содержимого каталога. Ключи: -F – добавляет символы типов, -a – показывает скрытые файлы (с .).

Утилита mkdir: создаёт новый каталог.

Утилиты копирования и перемещения: cp – копирует файлы, mv – перемещает/переименовывает.

Жёсткие ссылки: ещё одно имя файла (один inode). Создаются командой ln источник ссылка.

Создание файлов: touch имя (пустой файл) или перенаправление > имя.

Символьные ссылки: отдельный файл, содержащий путь к другому объекту. Создаются ln -s источник ссылка.

Удаление файлов и каталогов: rm – файлы, rmdir – пустые каталоги, rm -r – каталоги с содержимым. Восстановление невозможно (в Linux нет корзины для командной строки).

Назначение утилиты id: показывает UID, GID и список групп текущего пользователя.

Ярлыки объектов файловой системы (inode): содержат тип, права доступа, владельца, размер, временные метки, указатели на блоки данных.

Права доступа к файлу: r (чтение), w (запись), x (выполнение) для трёх категорий: владелец (u), группа (g), остальные (o).

Суперпользователь и его права: пользователь root с UID=0, имеет неограниченный доступ ко всем файлам и операциям.

Назначение утилиты sudo: позволяет выполнять команды от имени другого пользователя (обычно root) с логированием действий.

Утилиты поиска locate и find: locate – быстрый поиск по базе данных (требует updatedb), find – медленнее, но ищет в реальном времени и имеет больше возможностей для фильтрации.
